# 5. 404 信息的保存與傳播

人與政權的鬥爭，就是記憶與遺忘的鬥爭。

——米蘭·昆達拉《笑忘書》

讀到一篇好文章，存下來，在我們力所能及的範圍，和別人分享，也是一種抗爭。

——周保松《各人都盡一點力，突破新聞封鎖和言論審查 (opens new window)》

兄弟爬刪，各自努力

微信、微博、豆瓣等牆內平台上發佈的涉及敏感事件和話題往往會被管理者以「多人舉報」、「違反《網絡安全法》」等藉口刪除，即人們常說的「404」。本節將介紹幾種在網頁被「404」將之保存下來以便二次傳播的方法：

  • 網頁存檔
  • 導出 PDF
  • 網頁截圖
  • 文本轉錄

# 風險須知

備份/轉發前需注意提防【追蹤鏈接】（微博、推特、…）和【客戶端截圖盲水印】（豆瓣、知乎、…），鏈接追蹤參數和水印包含的用家 ID 等信息可被用於追查轉發者身份，追蹤參數還能收集所有瀏覽者的信息

作為應對，應學會區分【純淨鏈接】和【帶追蹤參數的鏈接】，以網易雲音樂為例
純淨版：https://y.music.163.com/m/song?id=000000000
id版：https://y.music.163.com/m/song?id=0000000000&userid=000000000
uct版：https://y.music.163.com/m/song?app_version=0.0.00&id=00000000&uct2=XXXXXXXXXXXXXXXXXXXXXXXXXXXXX

通用應對建議：
1 轉發前不要使用【客戶端】，改用【網頁版】
2 不要登錄【賬號】
3 使用 VPN/代理
4 盡量避免轉發【原鏈】和【截圖】，多用 Wayback 等網頁備份工具，轉發互聯網檔案館 web.archive.org 的存檔鏈接

長毛象用家可以關注長毛象上的 URL Bot https://bgme.me/@url

# 1 網頁存檔

在線網頁存檔工具的優勢在於：

  • 可以保持網頁的原貌，相比於文本轉錄在二次傳播時更具公信力
  • 網頁鏈接的形式更便於在社交網站和論壇傳播，比直接分享文件更便捷

# 1.1 web.archive.org

互聯網檔案館 (opens new window)（Internet Archive）是一個非營利性的的數字圖書館組織，它的網站時光機 (opens new window)（Wayback Machine）會用爬蟲自動抓取網頁，同時也提供了保存頁面功能。

用家只需在網站時光機網站首頁右下角的「Save Page Now」輸入網頁 URL 再按下「Save Page」按鈕即可保存網頁內容。

（已被 GFW 封鎖）

# 1.2 archive.today

archive.today (opens new window) 是一個私人資助的數字時間囊網站，提供抓取網頁內容的服務。擁有 archive.is、archive.li、archive.fo 等多個不同的域名，支援短鏈接形式轉發分享。

（已被 GFW 封鎖）

# 1.3 輔助工具

# 1.3.1 Wayback

Wayback (opens new window) 是一個自由開源、可自托管的集成式網頁存檔工具套件，支援將網頁存檔至 web.archive.org、archive.today、Telegraph 和 IPFS，還支援轉存為 PDF、HTML、HAR、WARC 和圖片。

使用方法 1：使用公共實例
https://wabarc.eu.org (opens new window)
https://initium.eu.org (opens new window)
wabarc/status (opens new window) 可以查看公共實例的在線狀態。
打開實例網站，在文本框內輸入需要存檔的源網頁鏈接，完成後點擊存檔按鈕即可。
Wayback 會自動將存檔結果添加到 https://loph.notion.site/f514a62ea1254d05ac5c92a48b9fe9b5 (opens new window)（信息來源：舊倉庫 Issue 區 (opens new window)） 存檔完成後請及時取回結果另行保存或二次發佈。

使用方法 2：使用 Telegram bot
@wabarc_bot (opens new window)

使用方法 3: 自托管
本地部署、在線部署以及更多使用場景請參閱 Wayback 倉庫的 README.md 文件。

# 1.3.2 Archiveror

Archiveror (opens new window) 是一個【瀏覽器插件】，支援 Firefox 和 Chrome，可以將網頁一鍵備份到 web.archive.org 和 archive.today 在內的多家網頁存檔網站。

# 1.3.3 archivenow

archivenow (opens new window) 是美國老道明大學網絡科學與數字圖書館研究組 (ODU WS-DL) 開發的能將網頁資源轉化為網頁存檔的【命令行工具】，支援 web.archive.org 和 archive.today 在內的多家網頁存檔網站。

# 2 導出PDF

絕大多數瀏覽器都支援將網頁導出為 PDF 保存。（File > Print > Save as PDF）

iOS 13 及更新的系統中，在 Safari 瀏覽器中對網頁截圖後，系統會提供將整個網頁保存為 PDF 的選項。

PDF 的分享方式：

  • 使用 OnionShare (opens new window) 分享
  • 上傳到匿名網盤/在線文件分享網站
    • https://anonfiles.com/ (opens new window)
      ⋯⋯
  • 上傳到 Telegram 頻道
  • 上傳到國外網盤，再分享網盤文件的鏈接
    • Google Drive
    • Dropbox ⋯⋯

*避免使用百度網盤等中國公司的服務，以此規避網絡審查與網絡實名制帶來的人身風險。

# 3 網頁截圖

【長截圖】工具可以用於保存新浪微博等難以使用在線網頁存檔工具抓取的內容。

# 3.1 網頁截圖/長截圖

3.1.1 移動端

iOS: Picsew, Tailor 等。
Android：PPIICC 等。

3.1.2 桌面端

Firefox 支援網頁截圖 (opens new window)。點擊右鍵 > 「截圖」 > 「截取整個頁面」

Chrome 用家可以利用 Chrome 開發者工具進行網頁截圖 (opens new window)。
macOS：
Command + Option + I
①截取整個網頁
Command + Shift + P
輸入命令：Capture full size screenshot
②（模擬移動設備）截取手機版網頁長圖
Command + Shift + M
點擊右上方的擴展按鈕選擇「Capture full size screenshot」

Windows： ①Control + Option + F12
截取整個網頁
②Control + Shift + P
輸入命令：Capture full size screenshot

# 3.2 截圖拼接

對於過長的截圖，【長截圖】工具可能無法一次性抓取，一些客戶端有時也會出現無法生成長圖的情況，此時可以採取分頁截圖後再拼接的方法。

# 3.3 長圖分享

分享長圖時應注意圖片可能會因壓縮變糊而無法閱讀。

  • 如通過 Telegram 分享長圖，建議選擇以文件/File 而非圖片形式發出
  • 如通過 Twitter 等 SNS 平台分享，建議將長圖上傳到不受中國審查的第三方圖床或網盤，分享圖片/文件的 URL 鏈接

# 4 文本轉錄

你還可以將涉及敏感話題的網頁內容拷貝後粘貼到支援以 URL 鏈接形式轉發的內容發佈平台。

  • 在線剪貼板工具

    • PrivateBin (opens new window)
      極簡、開源的在線剪貼板網站。網站本身不讀取記錄文本的內容，所有數據只在瀏覽器端以 256-bit AES 加密/解密。用家可以選擇剪貼板文本的保存時間（5分鐘、10分鐘、1小時、1天、1周、1個月、1年、永久）和文本格式（純文本、源碼、Markdown），可選擇閱後即焚、開放討論或者為鏈接添加密碼。
      公共實例列表 (opens new window)

  • Telegraph (opens new window)
    Telegram 提供的匿名博客服務。（已被 GFW 封鎖；請嘗試 Telegram 的替代域名，如 https://graph.org (opens new window) 和 https://te.legra.ph (opens new window)）

  • Matters (opens new window)
    基於區塊鏈、以 IPFS (opens new window) 存儲信息的內容發佈平台。IPFS 基於 P2P，難以被 GFW 徹底封殺。每篇 Matters 作品標題下都有「分布式入口」按鈕，開啓後可以查看「作品指紋」和「公共節點」。
    「指紋」是一篇作品上載到 IPFS 後生成的獨一無二的 ID，通過指紋可在 IPFS 不同節點調取作品內容。
    「公共節點」是一篇作品在 IPFS 網絡的存儲地點，你可以使用任意公共節點地址對作品進行傳播。
    Matters 域名已經被 GFW 封鎖，但是 IPFS 還有很多未被封鎖的公共節點，因此 Matters 作品的公共節點地址仍可在牆內訪問。

  • 不要使用有道雲筆記、石墨筆記等中國廠商提供的服務或者特供中國市場的服務，以及自帶敏感詞過濾的錘子便簽等【中資軟件】

  • duty-machine (opens new window)
